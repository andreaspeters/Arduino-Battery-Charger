Dieses Projekt ist im Rahmen der E15 Amateurfunk Bastelrunde entstanden.

## Allgemeine Notizen

- Versorgungsspannung 5 Volt
- Aufladen von AA und AAA Batterien
- Aufgrund der Tatsache das wir eine Solarzelle als Stromquelle nehmen, verwenden wir eine Impulsladung
- Maximal 1/10 Ampere der Batteriezelle
